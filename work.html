Grooup HTML Email

Type: University Project

As Part of my professional portfolio module at university I was given a task to create a HTML email for the company Grooup.

Grooup Html Email

This Project was undertaken as part of the Professional Portfolio Module at Edge Hill University

Planning

At the start of this project I firstly looked at what current HTML Emails look like, to do this I used a website called ReallyGoodEmails, I also looked within my own Email Mailbox to see how the companies that I am interested in format their emails.

From this I found that emails tend to follow the design of Logo at the top and images and text followed below, this can be seen by the example below

Once having an idea of what current HTML Emails look like I came up with some low fidelity prototypes

From the prototypes listed above I chose this design asI felt that this style is very similar to the Grooup website. Once chosen I then created the following high fidelity prototype to visualise what the email would actually look like with the appropiate content added.

After creating the high fidelity prototype I then had to start to think how to code html emails and as I had never created an HTML email before, I did some research and found that html tables are the best way to get the layout of HTML Emails to work within outlook, so it was important that I used tables as the main aim of the project was to get the email to work on all email clients.

Testing

After the email was finished it was tested using EmailOnAcid this shown whether the email would work on mutliple clients, the results shown that the email would work all 88 clients tested. The clients included the mail apps on both ios and android and gmail etc.
